Únete a la industria multibillonaria!

Programar es el proceso de crear un conjunto de instrucciones para decirle a una computadora cómo realizar una tarea. Se puede programar utilizando una variedad de lenguajes de programación de computadora, como JavaScript, Python y C++.

Fue en 1957 cuando el conocido científico John W. Backus creó Fortran, el primer lenguaje de programación conocido que tenía como fin realizar cálculos numéricos y científicos de manera eficiente.

La programación puede seguir muchos enfoques, o paradigmas, es decir, diversas maneras de formular la resolución de un problema dado. Algunos de los principales paradigmas de programación son: programación imperativa, programación declarativa, programación funcional, programación orientada a objetos, programación lógica y programación estructurada.
